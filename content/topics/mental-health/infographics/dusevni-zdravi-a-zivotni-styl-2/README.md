# Možnosti odborné léčby

Tato část se věnuje terapeutickým postupům. Rozebírá farmakoterapii, tedy nasazení různých typů antidepresiv či anxiolytik, a aktivně vyvrací běžné mýty a předsudky, které lidi od užívání psychofarmak často odrazují. 

Druhým hlavním pilířem je psychoterapie, a to zejména Kognitivně-behaviorální terapie (KBT), u které je podrobně popsáno, jak pomáhá pacientům měnit nefunkční vzorce myšlení a chování.
