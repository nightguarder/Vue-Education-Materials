# Role životního stylu a psychosomatika

Zdůrazňujeme celostní přístup k léčbě. Terapii je nezbytné podpořit zdravou životosprávou – texty vysvětlují konkrétní přínosy zdravého stravování, dostatku fyzické aktivity a kvalitní spánkové hygieny v prevenci i léčbě psychických onemocnění. 

Probírá se zde také psychosomatika, tedy to, jak se dlouhodobý stres, úzkosti a nezpracované emoce mohou promítat do nevysvětlitelných tělesných symptomů.
