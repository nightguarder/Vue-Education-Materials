# Léčba úzkosti a deprese: léky i biorytmy

Tato epizoda se zaměřuje na moderní přístupy k léčbě duševních onemocnění, které propojují klasickou medicínu s péčí o životní styl.

## Hlavní body:
- **Odborná léčba**: Jak funguje farmakoterapie (antidepresiva a anxiolytika) a proč se nebát vyhledat odborníka.
- **Vliv biorytmů**: Proč je pro psychické zdraví klíčový pravidelný režim, kvalitní spánek a soulad s přirozenými biorytmy těla.
- **Celostní přístup**: Propojení moderní psychiatrie s psychosomatikou a zdravou životosprávou.
